PySchool - Especificação de Banco de Dados (MySQL)

Especificação das tabelas, relacionamentos e schema do banco de dados local MySQL para a plataforma PySchool.

Resumo de Relacionamentos:
1. users 1:N user_progress
2. modules 1:N lessons
3. lessons 1:N exercises
4. users 1:N user_exercise_submissions
5. exercises 1:N user_exercise_submissions
6. users N:M achievements (via user_achievements)

Tabela: users

Armazena os dados dos alunos utilizando a plataforma localmente.

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
usernameVARCHAR(50)NOUNIQUE-Nome de login
password_hashVARCHAR(255)NO--Senha criptografada (bcrypt)
total_xpINTNO-DEFAULT 0Pontuação de experiência
created_atTIMESTAMPNO-DEFAULT CURRENT_TIMESTAMPData de criação

Tabela: modules

Trilhas de aprendizado principais (ex: Básico, Intermediário).

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
titleVARCHAR(100)NO--Nome do módulo
descriptionTEXTYES--Descrição do módulo
order_numINTNO--Ordem de exibição

Tabela: lessons

Lições dentro de cada módulo. Contém a teoria.

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
module_idINTNOFK-Ref: modules(id)
titleVARCHAR(100)NO--Título da lição
content_mdLONGTEXTNO--Conteúdo teórico em Markdown
order_numINTNO--Ordem dentro do módulo
xp_rewardINTNO-DEFAULT 50XP ganho ao concluir

Tabela: user_progress

Relaciona usuários às lições que eles concluíram.

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
user_idINTNOFK-Ref: users(id)
lesson_idINTNOFK-Ref: lessons(id)
completed_atTIMESTAMPNO-DEFAULT CURRENT_TIMESTAMPData de conclusão

Tabela: exercises

Desafios de código embutidos nas lições ou módulos.

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
lesson_idINTYESFK-Ref: lessons(id) - Pode ser nulo se for desafio avulso
promptTEXTNO--Enunciado do problema
starter_codeTEXTYES--Código inicial do usuário
test_scriptTEXTNO--Script de testes (unittest) para validar a resposta
xp_rewardINTNO-DEFAULT 100XP ganho ao resolver

Tabela: user_exercise_submissions

Histórico de tentativas de código dos usuários nos exercícios.

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
user_idINTNOFK-Ref: users(id)
exercise_idINTNOFK-Ref: exercises(id)
submitted_codeTEXTNO--Código enviado pelo usuário
is_correctBOOLEANNO-DEFAULT FALSESe passou nos testes ou não
created_atTIMESTAMPNO-DEFAULT CURRENT_TIMESTAMPData da tentativa

Tabela: achievements

Conquistas/Badges disponíveis no sistema.

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
nameVARCHAR(50)NO--Nome da conquista (ex: "Mestre da POO")
descriptionVARCHAR(255)NO--Como conquistar
icon_urlVARCHAR(255)YES--Caminho do ícone local

Tabela: user_achievements

Tabela de junção (N:M) entre usuários e conquistas.

CampoTipoNullKeyExtraDescrição
idINTNOPKAUTO_INCREMENTIdentificador único
user_idINTNOFK-Ref: users(id)
achievement_idINTNOFK-Ref: achievements(id)
awarded_atTIMESTAMPNO-DEFAULT CURRENT_TIMESTAMPData de recebimento
